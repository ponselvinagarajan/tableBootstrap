# HTML CSS DESCRIPTION
1 Parent Container Add width ,height only
2 Component Breathing space, add top - bottom space only
3 Container Device screen size,add max-width,margin-auto
4 Row Add display grid and padding left-right.divide the row if need how many columns.then use grid-template -column property
5 Column Align all content in center.use display-flex,align-item,justify-content property
6 SubColumn Align all content in center.use display-flex,align-item,justify-content property.if need to change the direction of content use flex direction
7 Content container Add needed style property
8 Content Add needed style property
